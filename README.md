# DE_Zoomcamp_2026
Main repository for the Data Engineering Zoomcamp 2026

## Module Definitions
I named the folders in a lazy manner so here are the core concepts and full title for each module of the data engineering zoomcamp:
 1. **Module 1 - Docker-Terraform** > key concepts include docker (containerization) and terraform (for setting up infrastructure via code).
 2. **Module 2 - Workflow-Orchestration** > key concepts include usage of kestra for harmonizing workflows, monitoring logs and even running workflows automatically.
 3. **Module 3 - Data-Warehousing** > key concepts tackle BigQuery, schema design, and efficient storage/query patterns.
 4. **Module 4 - Analytics-Engineering** > key concepts include usage of dbt (Data Build Tool) for modeling, transformations, and DuckDB for lightweight local analytics.
 5. **Module 5 - Data-Platforms** > key concepts include usage of Bruin, a data platform that handles asset lineage, dependencies, and orchestrates transformations across multiple data sources.
 6. **Module 6 - Batch Processing** > key concepts include Spark/PySpark.
